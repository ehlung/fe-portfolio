# Frontend Portfolio

빌드 없이 배포할 수 있는 정적 포트폴리오입니다.

## 수정 방법

1. `content.js`의 값을 실제 정보로 바꿉니다.
2. `index.html`을 브라우저에서 열어 화면을 확인합니다.
3. `fe-portfolio` 폴더를 Netlify, Vercel, GitHub Pages 중 하나에 배포합니다.

## 필요한 정보

- 이름 / 희망 직무 한 줄
- 자기소개 3-5문장
- 이메일, GitHub, 블로그나 Notion 링크
- 이력서 PDF 또는 공개 링크
- 대표 프로젝트 2-4개
- 각 프로젝트의 이름, 기간, 한 줄 설명, 담당 역할, 사용 기술, GitHub 링크, 배포 링크
- 교육, 부트캠프, 인턴, 동아리, 수상 등 경험
